# unturnedDS

Public download website for **unturnedDS — Unturned Dedicated Server**.

The Windows executable is distributed through GitHub Releases. The application creates and manages local Unturned dedicated servers and can store world snapshots in a user-selected cloud-synced folder.

On first launch the single EXE installs itself for the current Windows user and adds an `unturnedDS` shortcut to the Start Menu.

Steam credentials are never collected by unturnedDS. Steam verification uses the installed Steam client and local library manifests.
